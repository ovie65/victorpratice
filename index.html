2. write a bref history of JavaScript that covers all version up to ECMA6

JavaScript is a popular programming language used for creating interactive websites, web applications, and mobile applications. It was created by Brendan Eich at Netscape in 1995 and was originally called Mocha, then later changed to LiveScript before finally being named JavaScript. JavaScript quickly gained popularity due to its ability to add dynamic and interactive features to websites, making it a powerful tool for web developers. In 1997, Netscape submitted JavaScript to the European Computer Manufacturers Association (ECMA) for standardization,
resulting in the first version of the ECMA Script standard in 1997, which was also known as ES1. ES2, the second version of the ECMA Script standard, was released in 1998 and included new features such as try/catch error handling and regular expressions. ES3 was released in 1999 and added more features such as the ability to work with regular expressions, better string manipulation, and improved error handling. In 2000, ES4 was proposed, but it was never released due to disagreements among stakeholders. Instead, ES3.1 was released in 2008, which included new features such as getters and setters, array comprehensions, and destructuring assignments. ES5 was released in 2009 and added several new features, including strict mode, which provided a way to opt into a more secure and efficient subset of the language. It also introduced new array and object manipulation methods, such as map(), filter(), reduce(), and Object.keys(). ES6, also known as ECMAScript 2015, was released in 2015 and included significant improvements to the language. It introduced new features such as let and const for variable declarations, arrow functions, classes, modules, and template literals. It also included improvements to the syntax and semantics of the language, making it easier and more intuitive to use.
